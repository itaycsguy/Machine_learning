How To Use?
===========
1. Open CMD.
1.note. Check that the required python version is accessible through `python` environment variable.
2. Run: `python -m pip install -r requirements.txt` to install all needed packages.
2.note. If any problem is up - try to run the script anyway, otherwise talk to us.
3. Run: `cd ML_HW`
